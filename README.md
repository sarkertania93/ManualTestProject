# 🧪 Manual Software Testing Project – Startech

A comprehensive **Manual Software Testing Project** created for the **Startech** web application. This repository demonstrates the complete Software Testing Life Cycle (STLC), including planning, test design, execution, reporting, and test metrics.

---

## 📌 Project Overview

This project showcases end-to-end manual testing artifacts prepared for the **Startech** application. It covers requirement analysis, test planning, scenario creation, test case design, bug reporting, test execution, and reporting.

The objective of this project is to demonstrate practical knowledge of manual software testing and industry-standard QA documentation.
